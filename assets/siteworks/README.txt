SiteWorks screenshots
=====================

These images are used in the "SiteWorks" section on the Services page of the
website (index.html). They were pulled from SiteWorks-Pitch-Deck.pdf.

Current files (referenced by index.html):
  dashboard.jpg     -> hero image (Sales Dashboard)
  work-orders.jpg   -> gallery: Work orders & live status
  scheduling.jpg    -> gallery: Scheduling & dispatch
  invoicing.jpg     -> gallery: Invoicing

To swap in a newer screenshot, just replace the file here using the SAME
filename (keep it .jpg) and commit — the website will pick it up automatically.
If you ever rename a file, update the matching src="assets/siteworks/..." line
in index.html to match.

Note: these screenshots show SiteWorks demo/sample data (example customer and
tech names). If you'd rather show different data, retake the screenshots and
replace the files here.
